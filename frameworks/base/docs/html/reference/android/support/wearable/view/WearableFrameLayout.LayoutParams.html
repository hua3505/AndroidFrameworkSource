WearableFrameLayout.LayoutParams

WearableFrameLayout.LayoutParams (int width,
                int height,
                int gravity,
                int widthRound,
                int heightRound,
                int gravityRound)

Creates a new set of layout parameters with the specified width, height and weight for both square and round devices.

Parameters
width int: the width on a square device, either MATCH_PARENT, WRAP_CONTENT or a fixed size in pixels
height int: the height, either MATCH_PARENT, WRAP_CONTENT or a fixed size in pixels
gravity int: the gravity
widthRound int: the width override for a round device, either MATCH_PARENT, WRAP_CONTENT or a fixed size in pixels
heightRound int: the height override for a round device, either MATCH_PARENT, WRAP_CONTENT or a fixed size in pixels
gravityRound int: the gravity override for a round device
